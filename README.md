# Illuminating the black-box: An explainability study (XAI) of a Deep CNN for near-infrared spectroscopy

The code for the paper "Illuminating the black-box: An explainability study (XAI) of a Deep CNN for near-infrared spectroscopy". 


## Environment management
To create a base environment: 
```bash
conda env create -f environment.yml

```

To launch the notebooks :
```bash
conda activate xai
jupyter lab .
```
